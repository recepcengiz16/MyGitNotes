Uzak sunucuya bağlanma ve projeyi push yapma

git remote add origin https://github.com/recepcengiz16/MyGitNotes.git => oluşturduğum repository url bağlantısını ekle demiş olduk. origin diyerek de isim vermiş oluyoruz, fark etmez başka bir şey de diyebiliriz. origin main, origin dev(developer ortamı için oluşturduğum branchin adı olabilir.)

daha sonra git remote diyerek origin adında bir klasör oluşturmuş oluyoruz localde

git push -u origin main => push ile repositoryimize projeyi atmamızı sağlıyoruz. origin de vermiş olduğumuz isim olmuş oluyor yukarıdaki, daha sonra da hangi branche göndereceğiz main branchine

git remote remove origin => ile de origini silmemizi sağlıyoruz. Yani remote origin bağlantısını kaldırıyoruz.

git pull => githubdan dosyaları çekmek için

git clone https://egvrcn@bitbucket.org/egvrcn/test.git(proje yolu) =>  Var Olan Bir Remote Projeye Entegrasyon. İlk defa uzaktan bir projeye bağlanıp onun git halini almak için aşağıdaki kod yazılır. Bu sayede ilgili git projesi son haliyle yüklenir ve otomatik olarak origin remote'u eklenir.
