Git Commit nedir?

Local repository ye göndermeden önce projeyi staging areaya göndermemiz gerekiyor, daha sonra commit yaparak local repositorye göndermemiz gerekli

git add index.html => tek dosyayı bu şekilde staging areaya eklemiş oluyoruz.

git add . => staging areaya eklenmeyen bütün dosyaları al, staging areaya ekle demiş oluyoruz. Ve commitlemeye hazır hale gelmiş oluyor

git commit -m "İlk mesaj" => mesajla birlikte local repositorye göndermiş oluyoruz.
